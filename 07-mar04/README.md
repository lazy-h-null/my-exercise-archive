# Python & DaisyUI Web Generator Project

This project focuses on Python-based HTML automation. It reads a `base.html` template and generates a styled `index.html` using DaisyUI components.

## 🎯 Assignment Goal
Modify `html_editing.py` to add new elements from DaisyUI.
Use the `</body>` tag as a reference to replace and add elements on top of it.

## ✅ Assignment Checklist
- [x] 1. Define new elements as variables
- [x] 2. Copy DaisyUI code and paste into variables
- [x] 3. Use multiline strings (`""" ... """`)
- [x] 4. Share the website via GitHub Pages
