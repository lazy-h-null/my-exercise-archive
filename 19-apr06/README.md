# 🦄💚🎸 Bistro Project & Django Intro (Class 19)

An introductory web application project to learn **Django**, a high-level Python web framework. This project focuses on the **MVT (Model-View-Template)** architecture using a "Restaurant" metaphor.

## 🍽️ The Restaurant Metaphor

| Component | Role | Metaphor |
| :--- | :--- | :--- |
| **URL** | The Host | Directs the customer to the right table |
| **View** | The Waiter | Takes orders and communicates with the kitchen |
| **Model** | The Pantry | Where all ingredients (data) are stored |
| **Template** | The Plate | How the food is presented to the customer |
